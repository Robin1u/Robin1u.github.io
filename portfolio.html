精选 Web 应用 全栈

校园互联

作为 2025 年的一次黑客松项目，"校园互联" 解决了大学生一个真实的痛点：如何找到合适的学习伙伴。平台允许学生创建个人档案、登记所修课程，并为他们匹配志同道合的学习搭档。短短 3 个月内，产品自然增长至两所校园共 1200 名用户。

最大的挑战是实时消息。我用 Node.js 实现了一套轻量 WebSocket 方案，不依赖完整的基础设施也可以稳定扩展。匹配算法基于课程重合度、可用时间和学习风格偏好进行加权打分。

技术栈
HTML / CSS / JS Node.js Express WebSocket SQLite

UI / 设计CSS

Tempo 设计系统

Tempo 是我为加速原型设计流程而构建的个人设计系统。它包含一个拥有 60+ 组件的 Figma 库、一套匹配的 CSS 变量体系和一个文档站点。核心理念：深色优先、高对比度、去除不必要的装饰。

每个组件都内置了无障碍支持 —— 合理的 ARIA 角色、完整的键盘导航，以及符合 WCAG AA 标准的色彩对比度。这套系统目前是我另外三个项目的基础。

技术栈
Figma CSS 变量 原生 JS HTML
